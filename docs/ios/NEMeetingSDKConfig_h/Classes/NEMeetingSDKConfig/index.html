reuseNIM

@property (nonatomic,
    assign) BOOL reuseNIM;
Discussion

配置是否复用底层NIM-SDK的长连接通道。 当启用该配置时：

1. 在进行Meeting-SDK登录时，需要保证NIM-SDK已经登录，且Meeting账号与已登录的NIM账号关联关系匹配， 复用才能成功，否则复用失败。

2. 在会议过程中，应用需要保证NIM处于登录状态，否则会造成会控失败。

仅当应用中接入了Meeting-SDK，同时还需独立接入和使用NIM-SDK，才需要开启该配置；其他情况下请忽略该配置。

useAssetServerConfig

@property (nonatomic,
    assign) BOOL useAssetServerConfig;
Discussion

是否检查并使用私有化服务器配置文件nim_server.conf(固定命名），默认为FALSE。

在私有化项目中需要开启此选项，将配置文件重命名为 nim_server.conf，添加到项目根目录下，与Info.plist文件同级目录。
