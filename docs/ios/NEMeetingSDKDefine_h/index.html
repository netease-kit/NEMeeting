NECallbackOnResult

typedef void(^NECallbackOnResult)(
    NSInteger resultCode,
    NSString *resultMsg,
    id resultData);
Parameters
resultCode

错误码

resultMsg

错误描述，可能为空

resultData

返回值，数据类型为id，可能为空

Discussion

SDK API的通用回调接口。会议SDK提供的接口多为异步实现，在调用这些接口时，需要提供一个该接口的实现作为回调参数。 异步调用结束时，回调该方法通知调用方。

    -5,   MEETING_ERROR_FAILED_ALREADY_IN_MEETING = -6,   MEETING_ERROR_FAILED_LOGIN_ON_OTHER_DEVICE = 
    -7,   ERROR_MEETING_PASSWORD_ERROR = -8,   ERROR_CODE_CANCELLED = -9,   ERROR_CODE_ALREADY_OPEN_CONTROL = 
    -10,   ERROR_CODE_UNEXPECTED = -100000,   ERROR_CODE_NOT_IMPLEMENTED = 
    -100001,   ERROR_CODE_SDK_UNINITIALIZE = -100002,

Constants
ERROR_CODE_SUCCESS

对应接口调用成功

ERROR_CODE_PARAM_ERROR

接口调用失败，原因为参数错误

ERROR_CODE_FAILED

对应接口调用失败

MEETING_ERROR_FAILED_IM_LOGIN_ERROR

SDK登录鉴权请求失败

MEETING_ERROR_NO_NETWORK

接口调用失败，原因为无网络连接

MEETING_ERROR_FAILED_MEETING_ALREADY_EXIST

创建会议失败，原因为当前已经存在一个使用相同会议ID的会议。 调用方此时可以加入该会议或者等待该会议结束后重新创建

MEETING_ERROR_FAILED_PARAM_ERROR

接口调用失败，原因为参数错误

MEETING_ERROR_FAILED_ALREADY_IN_MEETING

创建或加入会议失败，原因为当前已经处于一个会议中

MEETING_ERROR_FAILED_LOGIN_ON_OTHER_DEVICE

创建或加入会议失败，原因为当前登录状态已失效

ERROR_MEETING_PASSWORD_ERROR

加入会议失败，原因为会议密码错误

ERROR_CODE_CANCELLED

加入会议失败，原因为主动取消加入

ERROR_CODE_ALREADY_OPEN_CONTROL

遥控器打开状态，不能进入会议；会议非idle，遥控器不能使用

ERROR_CODE_UNEXPECTED

接口调用失败，原因为出现了异常

ERROR_CODE_NOT_IMPLEMENTED

接口调用失败，原因为对应接口未实现

ERROR_CODE_SDK_UNINITIALIZE

接口调用失败，原因为SDK未初始化或初始化失败

Discussion

SDK通用错误码与错误描述
