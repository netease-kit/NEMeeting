NEMenuStateController

typedef void(^NEMenuStateController)(
    BOOL transition,
    _Nullable id extra);
Parameters
transition

是否进行状态迁移

extra

预留参数，当前可空

Discussion

自定义菜单的状态迁移控制器回调。当自定义菜单项被点击时，SDK会回调对应接口，并提供该回调用于控制菜单项是否应该进行状态迁移。
