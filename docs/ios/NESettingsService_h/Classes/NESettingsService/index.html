Introduction

会议设置服务，可设置入会时、会议中的一些配置信息

如入会时的音视频开关选项，如果在入会时未指定NEMeetingOptions，则使用该设置服务提供的默认值

该设置服务使用设备本地存储，暂不支持漫游
